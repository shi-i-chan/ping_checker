# Ping checker


1) Fill config `config.py`.

<ul>

```python
fn = 'latency.csv'
save_every = 10  # n interations
sleep = 10  # n seconds

urls = {
    'google': 'google.com',
    'wiki': 'wikipedia.org',
    'youtube': 'youtube.com',
}

```

</ul>

2) Run `start_pinger.bat`.

<ul>

Files `latency.png` and `latency.csv` will appear in root folder.
